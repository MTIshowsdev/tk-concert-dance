## Editing with Claude

This file is hosted so it can be read and edited by Claude across conversations.

**To make changes:**

1. Paste this raw URL into a Claude conversation:
   `https://raw.githubusercontent.com/MTIshowsdev/tk-concert-dance/main/index.html`
2. Tell Claude what to change
3. Copy Claude's output and paste it back here via the "Edit this file" pencil icon on the `index.html` page

Changes published to `main` will appear at the live preview URL within ~1 minute.
